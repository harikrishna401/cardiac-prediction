# ❤️ Cardiac Disease Prediction System

## 📌 Project Overview

This project predicts whether a person is at risk of **cardiac (heart) disease** using machine learning.
The model is trained on medical parameters such as age, blood pressure, cholesterol, etc.

---

## 🚀 Features

* Predicts heart disease risk
* Simple user input
* Machine Learning based model
* Fast prediction
* Easy to use

---

## 🧠 Machine Learning Algorithm

* Logistic Regression / Random Forest / (your model name)
* Trained using medical dataset
* Accuracy based prediction

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/harikrishna401/cardiac-prediction.git
```

Go to project folder:

```
cd cardiac-prediction
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```
python app.py
```
